Como validar um site

Passo 1: Clique no botão Validar um site localizado no cabeçalho.

Passo 2: Insira o endereço (URL) do site que deseja validar no campo indicado.

Passo 3: Clique em Enviar para iniciar a validação.

Passo 4: Analise o resultado apresentado, incluindo pré-visualização e status.

Passo 5: Caso o site esteja correto, confirme e preencha o formulário de avaliação.

Passo 6: Complete todos os campos obrigatórios, incluindo título, email, status, issues, feature request, task e DS.

Passo 7: Clique em Validar para enviar sua avaliação.

Observação: Certifique-se de revisar todas as informações antes de enviar. O resultado será encaminhado ao responsável pelo projeto.

Como gerar link Validate

Passo 1: Clique no botão Gerar link Validate no cabeçalho.

Passo 2: Preencha os campos solicitados: link do site, nome do site, seu nome, email e tópico.

Passo 3: Escolha as opções de retorno desejadas (issues, feature request, task, DS).

Passo 4: Clique em Gerar para criar o link personalizado.

Passo 5: O link gerado será exibido. Utilize o botão Copiar link para compartilhar.

Observação: O link Validate permite que outros usuários avaliem o site de forma rápida e organizada, com campos pré-configurados conforme suas escolhas.
